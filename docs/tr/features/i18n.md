# Çoklu Dil Desteği (i18n)

Bunshelf, birden çok dilde dokümantasyon oluşturmayı kolaylaştıran yerleşik uluslararasılaştırma (i18n) desteğine sahiptir.

## Yapılandırma

### Dilleri Etkinleştirme

`docs/config.yaml` dosyasına diller ekleyin:

```yaml
defaultLocale: en
locales:
  en:
    indexPage: /intro
    localePrefix: english
  tr:
    indexPage: /intro
    localePrefix: turkish
  de:
    indexPage: /intro
    localePrefix: german
```

### Dil Seçenekleri

Her dil için şu ayarlar yapılabilir:

| Seçenek | Tür | Varsayılan | Açıklama |
|---------|-----|------------|----------|
| `indexPage` | string | "/intro" | Bu dil için ana sayfa yolu |
| `localePrefix` | string | dil kodu | URL öneki (ör. `tr` yerine `turkish`) |

### Dil Formatı

ISO 639-1 dil kodlarını kullanın:

| Kod | Dil |
|-----|-----|
| `en` | İngilizce |
| `tr` | Türkçe |
| `de` | Almanca |
| `fr` | Fransızca |
| `es` | İspanyolca |
| `ja` | Japonca |
| `zh` | Çince |

## Dizin Yapısı

İçeriği dile göre düzenleyin:

```
docs/
├── config.yaml
├── en/
│   ├── intro.md
│   └── getting-started/
│       └── installation.md
├── tr/
│   ├── intro.md
│   └── getting-started/
│       └── installation.md
└── de/
    └── intro.md
```

## URL Yapısı

### Varsayılan Dil (en)

İngilizce içerik kök dizinde sunulur (önek yok):

- `/intro` → `docs/en/intro.md`
- `/getting-started/installation` → `docs/en/getting-started/installation.md`

### Diğer Diller

Diğer diller URL'de `localePrefix` kullanır:

- `/turkish/intro` → `docs/tr/intro.md`
- `/german/intro` → `docs/de/intro.md`

::: tip Not
`localePrefix` özelleştirilebilir. Örneğin, `tr` yerine `turkish` kullanmak URL'leri daha okunabilir hale getirir.
:::

## Dil Değiştirici

Kullanıcılar kenar çubuğundaki dil değiştiriciyi kullanarak dil değiştirebilirler:

- Mevcut tüm dilleri gösterir
- Mevcut sayfa yolunu korur
- Seçilen dilde aynı sayfaya yönlendirir

## Dil Başına Kenar Çubuğu

Her dil için farklı kenar çubukları yapılandırın. `href` değerlerine dil öneki eklemenize gerek yok - otomatik eklenir:

```yaml
sidebar:
  en:
    - label: Getting Started
      items:
        - label: Introduction
          href: /intro
        - label: Installation
          href: /getting-started/installation
  tr:
    - label: Başlarken
      items:
        - label: Giriş
          href: /intro
        - label: Kurulum
          href: /getting-started/installation
```

::: tip Otomatik Önek
Varsayılan olmayan diller için `localePrefix` kenar çubuğu href'lerine otomatik eklenir. Yani `/intro`, Türkçe için `/turkish/intro` olur.
:::

## En İyi Uygulamalar

### İçerik Organizasyonu

- Tüm dillerde aynı yapıyı koruyun
- Sadece bazı değil, tüm sayfaları çevirin
- Yeni içerik eklerken tüm dilleri güncelleyin

### Çeviri Kalitesi

- Çeviriler için ana dil konuşanları kullanın
- Çevirileri marka sesinizle tutarlı tutun
- Teknik doğruluk için çevirileri gözden geçirin

::: tip İpucu
Tutarlı kenar çubuğu sıralamasını korumak için diller arasında aynı `order` değerlerini kullanın.
:::
